> A collection of Big Data labs for the **DS200** course @ **University of Information Technology (UIT)**.  
> Built for practical learning with distributed data processing, scalable pipelines, and analytics workflows.

## Course Focus

- Distributed data processing with Hadoop, Apache Spark
- Data ingestion and streaming pipelines
- ETL design for large-scale datasets
- Query optimization and performance tuning
- Batch vs. real-time data workflows

## Tech Highlights

- Spark for distributed computation
- Hadoop ecosystem concepts (HDFS)

## Goal

This repository is used to practice hands-on Big Data concepts from classroom theory to real workloads, with a focus on implementation, experimentation, and reproducibility.
